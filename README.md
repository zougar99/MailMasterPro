# 📧 MailMasterPro

> 📧 Professional Email Marketing Platform with AI Tools, Analytics, Campaigns, and Marketing Automation. Built with Node.js, Express, and Socket.io.

---

## ✨ Features

- ✔ **Bulk email sending with multi-SMTP**
- ✔ **HTML support with personalization**
- ✔ **Real-time tracking and analytics**
- ✔ Lightweight and optimized
- ✔ Cross-platform compatibility
- ✔ Regular updates

---

## 💻 Tech Stack

| Component | Technology |
|-----------|-----------|
| Language | Python |
| UI | CustomTkinter / PyQt |
| Platform | Windows / Linux / macOS |

---

## 🚀 Installation

```bash
git clone https://github.com/zougar99/MailMasterPro.git
cd MailMasterPro
pip install -r requirements.txt
```

---

## 🧰 Usage

```bash
python main.py
```
