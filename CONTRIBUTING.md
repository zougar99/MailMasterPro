# 🤝 Contributing to Email Marketing Pro

Thank you for considering contributing to this project! 🎉

## 📋 How to Contribute

### 1. 🍴 Fork the Project
Click the "Fork" button at the top right of this page.

### 2. 🌿 Create your Feature Branch
```bash
git checkout -b feature/AmazingNewFeature
```

### 3. ✅ Commit your Changes
```bash
git commit -m 'Add: ✨ Amazing New Feature'
```

### 4. 📤 Push to the Branch
```bash
git push origin feature/AmazingNewFeature
```

### 5. 🎯 Submit a Pull Request
Open a Pull Request and describe your changes!

## 🐛 Reporting Bugs

Please report bugs using the **Issues** tab. Include:
- Description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Screenshots (if applicable)

## 💡 Feature Requests

Open an **Issue** with the tag `enhancement` for new features.

## 📝 Code Style

- Use consistent indentation (2 or 4 spaces)
- Add comments for complex logic
- Follow JavaScript naming conventions

## 🧪 Testing

Test your changes before submitting:
```bash
npm test
```
